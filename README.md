## Run the application

In the project directory, run the following commands in sequence:

### `npm run seed`

Runs seeder script to create `admin@admin.com/test` and `user@user.com/test` demo users and demo roles

### `npm run server`

Runs the api server.<br />

### `npm run client`

Runs the client on the browser.<br/>
